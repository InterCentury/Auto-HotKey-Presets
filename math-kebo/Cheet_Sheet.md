### CORE IDEA

* Numbers → powers (², ³, ⁴…)
* Letters → superscript letters (ᵃ, ᵇ, ᶜ…)
* Hold **Right Alt** → activate power mode

### MINI MEMORY TRICK

* Numbers = instant exponents
* Letters = variable powers
* Hold RAlt = “power typing mode”
